meteor-directory-builder

A bash script that builds a directory structure suited for meteor apps.

Usage:

./mdb.sh (destination) // leave destination blank to build the directories in the same directory as mdb.sh

Installation: Add the following lines to your ~/.bashrc file to use mdb from anywhere:

# meteor-directory-builder alias alias mdb=(replace this with the path to mdb.sh)
